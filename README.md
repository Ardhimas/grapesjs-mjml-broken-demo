# grapesjs-mjml issue demo

This project outlines an issue where the grapesjs editor works in dev but not in prod.

## Initialize

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## Scripts

### Start project in dev

```bash
npm start
```

### Build project

```bash
npm run build
```

### Build and deploy prod build to local web server

```bash
npm run serve
```
